# ISHA-Integrated-Safety-System
An IoT-based emergency detection system utilizing Raspberry Pi 5 and YOLOv8 for home safety
### Project Overview
[cite_start]ISHA is an integrated emergency detection and alert system designed to monitor multiple household hazards, including fire, gas leaks, and earthquakes[cite: 632].

### Key Features
* [cite_start]**Multi-Hazard Detection:** Real-time monitoring of gas leaks (MQ-2), seismic activity (SW-420), and temperature/humidity (DHT22)[cite: 647, 651].
* [cite_start]**AI-Powered Fire Prevention:** Uses a Raspberry Pi Camera Module 3 and the YOLOv8n algorithm for real-time fire detection[cite: 649, 659].
* [cite_start]**Automated Mitigation:** Automatically activates a water mist mechanism upon fire detection[cite: 660, 691].
* [cite_start]**Mobile Integration:** Real-time alerts and live camera feed via a Kotlin-based Android application[cite: 327, 661].

### Hardware Stack
* [cite_start]Raspberry Pi 5 (Central Processing Unit) [cite: 652]
* [cite_start]Raspberry Pi Camera Module 3 [cite: 649]
* [cite_start]Sensors: MQ-2 (Gas), SW-420 (Vibration), DHT22 (Temp/Humidity) [cite: 650, 651]
* [cite_start]Actuators: MG996R Servos, Water Pump, Buzzer [cite: 660]
